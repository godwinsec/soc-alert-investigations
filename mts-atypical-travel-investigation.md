# SOC Alert Investigations Portfolio

This repository contains Tier-1 SOC alert investigations performed in a simulated Managed Security Service Provider (MSSP) environment.

## Environment Overview
- Role: Tier-1 SOC Analyst
- Platform(s): Microsoft Defender XDR, Microsoft Entra ID Protection
- Scenario: Simulated MSSP supporting multiple clients
- Data Source: Lab environment (non-production)

## Investigation Methodology
Each investigation follows a standard SOC workflow:
1. Alert review and scoping
2. Evidence collection and validation
3. Contextual analysis using client-specific constraints
4. Risk assessment and classification
5. Documentation and recommended actions

## Disclaimer
All users, IP addresses, and organizations referenced are part of a controlled training environment and do not represent real incidents.
